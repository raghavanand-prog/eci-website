# 🗳️ Election Commission of India (ECI) Web Portal

## 📌 Project Description

The Election Commission of India (ECI) Web Portal is a responsive web application developed using HTML, CSS, and JavaScript. It provides information about the Election Commission of India, the election process, voter awareness, and contact details. The project also includes an interactive voter registration system with form validation, Local Storage, and HTML5 APIs.

## 📋 Registration Module

The voter registration page includes:

- Full Name
- Email
- Password
- Mobile Number
- Date of Birth
- Gender Selection
- State Dropdown
- Identity Proof Upload
- Address
- Terms & Conditions Checkbox

After successful registration:

- Input is validated
- Details are displayed in a registration card
- Data is stored using Local Storage
- Success message is displayed

---

## 🌐 HTML5 APIs Used

### Geolocation API

Allows users to retrieve their current location.

### Local Storage API

Stores voter registration details in the browser even after the page is refreshed.
